# Derden

Deze sectie bevat verschillende aanvullingen van derden die exclusief (of mainly) zijn geschreven voor het gebruik samen met ASF. Deze variëren van ASF-plug-ins, tot eenvoudige webapplicaties en interne bibliotheken voor integratie, eindigend met volledige bots voor andere platforms. Als je iets aan de lijst wilt toevoegen, laat het ons weten op Discord of op onze Steam groep.

Houd er rekening mee dat onderstaande programma's **niet** onderhouden worden door ASF ontwikkelaars en daarom **we geen garantie geven over één van hen**vooral op het gebied van veiligheid, veiligheid of naleving van Steam ToS. Deze lijst wordt alleen als referentie gehandhaafd. Je moet er altijd voor zorgen dat de hulpbron van de derde partij die je gaat gebruiken veilig is en genoeg is voor jou, omdat je ze allemaal op eigen risico gebruikt.

## ASF plugins

### **[CatPoweredPlugins](https://github.com/CatPoweredPlugins)** (**[Rudokhvist](https://github.com/Rudokhvist)**)

- **[ASFAievementManager](https://github.com/CatPoweredPlugins/ASFAchievementManager)**, plugin voor ASF waarmee je Steam achievements kunt beheren.
- **[Verjaardag Plugin](https://github.com/CatPoweredPlugins/BirthdayPlugin)**, plugin voor ASF om verjaardagsgefeliciteerd te worden.
- **[CaseInsensitiveASF](https://github.com/CatPoweredPlugins/CaseInsensitiveASF)**, plugin voor ASF om botnamen hoofdletterongevoelig te maken.
- **[CommandAliasPlugin](https://github.com/CatPoweredPlugins/CommandAliasPlugin)**, plugin voor ASF om aangepaste commando-aliases voor ASF commando's te installeren.
- **[CommandlessRedeem](https://github.com/CatPoweredPlugins/CommandlessRedeem)**, plugin voor ASF om de sleutel opnieuw te implementeren zonder opdracht.
- **[ItemDispenser](https://github.com/CatPoweredPlugins/ItemDispenser)**, plugin voor ASF om automatisch handelsverzoeken te accepteren voor bepaalde type items.
- **[SelectiveLootAndTransferPlugin](https://github.com/CatPoweredPlugins/SelectiveLootAndTransferPlugin)**, plugin voor ASF die geavanceerde `overdracht` opdracht geeft voor het overdragen van Steam items.

### **[Citrineren](https://github.com/Citrinate)**

- **[BoosterManager](https://github.com/Citrinate/BoosterManager)**, ASF biedt een eenvoudig te gebruiken interface voor het omzetten van edelstenen in boosterpakketten, evenals diverse functies voor het beheer van inventarisaties en marktlijsten.
- **[CS2Interface](https://github.com/Citrinate/CS2Interface)**, plugin voor ASF waarmee u interactie kunt onderhouden met Counter-Strike 2 met **[IPC](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/IPC)**.
- **[FreePackages](https://github.com/Citrinate/FreePackages)**, plugin voor ASF die gratis pakketten op Steam vindt en deze aan je account toevoegt.

### **[Vita](https://github.com/ezhevita)**

- **[VriendAccepter](https://github.com/ezhevita/FriendAccepter)**, plugin voor ASF om automatisch alle vriendsuitnodigingen te accepteren.
- **[GameRemover](https://github.com/ezhevita/GameRemover)**, plugin voor ASF implementeren van een commando om Steam licenties te verwijderen voor geselecteerde bot instanties.
- **[GetEmail](https://github.com/ezhevita/GetEmail)**, plugin voor ASF het implementeren van een commando om het e-mailadres van de opgegeven bot instanties rechtstreeks van Steam op te halen.
- **[ResetAPIKey](https://github.com/ezhevita/ResetAPIKey)**, plugin voor ASF implementeren van een commando om de API key te resetten voor geselecteerde bot instanties.

### Overige

- **[ASFEnhance](https://github.com/chr233/ASFEnhance)**, plugin voor het verbeteren ervan met verschillende nieuwe functies, vooral commando's.
- **[ASFFreeGames](https://github.com/maxisoft/ASFFreeGames)**, plugin voor ASF waarmee deze automatisch gratis stoom spellen kan verzamelen die geplaatst zijn op reddit.

## Integraties

- **[ASFBot](https://github.com/dmcallejo/ASFBot)**, telegram bot geschreven in python met ASF integratie.
- **[ASF STM userscript](https://greasyfork.org/en/scripts/404754-asf-stm)**, voor degenen die geautomatiseerde handelsaanbiedingen willen versturen naar bots in onze **[ASF STM listing](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/ItemsMatcherPlugin#publiclisting)** via webbrowser, zonder de `MatchActief` functie te gebruiken, aangeboden door ASF.
- **[simple-asf-bot](https://github.com/deluxghost/simple-asf-bot)**, een andere (minimale) telegram bot geschreven in python met ASF-integratie.

## Hulpmiddelen

- **[CS2Script](https://github.com/Citrinate/CS2Script)**, stelt u in staat om uw Counter-Strike 2-opslageenheden te beheren met behulp van **[CS2Interface](https://github.com/Citrinate/CS2Interface)** plugin.
- **[Keys extractor](https://umaim.github.io/SKE)**, geeft je de mogelijkheid om toetsen te kopiëren en plakken in verschillende formaten en een `inwisselopdracht te maken` voor ASF. Bekijk **[GitHub repo](https://github.com/PixvIO/SKE)** voor meer details.
- **[ASF Mass Config Editor](https://github.com/genesix-eu/ASF_MCE)**, waarmee u meerdere ASF configuraties makkelijker kunt beheren.
