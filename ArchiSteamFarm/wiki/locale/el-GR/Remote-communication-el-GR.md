# Απομακρυσμένη επικοινωνία

Αυτή η ενότητα αναπτύσσει την απομακρυσμένη επικοινωνία που περιλαμβάνει η ASF, συμπεριλαμβανομένης περαιτέρω εξήγησης σχετικά με το πώς μπορεί κανείς να την επηρεάσει. Ενώ δεν θεωρούμε τίποτα παρακάτω ως κακόβουλο ή άλλως ανεπιθύμητο, ούτε είμαστε υποχρεωμένοι να το αποκαλύψουμε, θέλουμε να κατανοήσετε καλύτερα τη λειτουργικότητα του προγράμματος, ειδικά σε σχέση με το απόρρητο και τα δεδομένα σας που μοιράζονται.

## Ατμός

Το ASF επικοινωνεί με το δίκτυο Steam (**[διακομιστές CM](https://api.steampowered.com/ISteamDirectory/GetCMList/v1?cellid=0)**), καθώς και **[Steam API](https://steamcommunity.com/dev)**, **[Κατάστημα Steam](https://store.steampowered.com)** και **[κοινότητα Steam](https://steamcommunity.com)**.

Δεν είναι δυνατή η απενεργοποίηση οποιασδήποτε από τις παραπάνω επικοινωνίες, καθώς είναι το βασικό θεμέλιο του ASF βασίζεται στην παροχή της βασικής λειτουργικότητας του. Θα χρειαστεί να απέχετε από τη χρήση του ASF αν δεν είστε άνετα με τα παραπάνω.

## Ομάδα ατμού

Το ASF επικοινωνεί με την ομάδα **[Steam](https://steamcommunity.com/groups/archiasf)**. Η ομάδα σας παρέχει ανακοινώσεις, ειδικά νέες εκδόσεις, κρίσιμα θέματα, προβλήματα Steam και άλλα πράγματα που είναι σημαντικά για να κρατήσει την κοινότητα ενημερωμένη. Σας επιτρέπει επίσης να χρησιμοποιήσετε την τεχνική μας υποστήριξη, υποβάλλοντας ερωτήσεις, επιλύοντας προβλήματα, αναφέροντας ζητήματα ή προτείνοντας βελτιώσεις. Από προεπιλογή, οι λογαριασμοί που χρησιμοποιούνται στο ASF θα ενταχθούν αυτόματα στην ομάδα κατά τη σύνδεση.

Μπορείτε να αποφασίσετε να εξαιρεθείτε από την ένταξη στην ομάδα απενεργοποιώντας τη σημαία `SteamGroup` στο **[`RemoteCommunication`](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Configuration#remotecommunication)** ρυθμίσεις.

## GitHub

Το ASF επικοινωνεί με το API **[του GitHub](https://api.github.com)** προκειμένου να φέρει **[εκδόσεις ASF](https://github.com/JustArchiNET/ArchiSteamFarm/releases)** για τη λειτουργικότητα της ενημέρωσης. Αυτό γίνεται ως μέρος των αυτόματων ενημερώσεων (αν έχετε κρατήσει **[`UpdatePeriod`](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Configuration#updateperiod)** ενεργοποιημένη), καθώς και `ενημέρωση` **[εντολή](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Commands)**. Μπορείτε να επηρεάσετε την επικοινωνία του ASF με το GitHub μέσω **[`UpdateChannel`](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Configuration#updatechannel)** ιδιότητα - ρύθμιση σε `Κανένα` δεν θα έχει ως αποτέλεσμα την απενεργοποίηση ολόκληρης της λειτουργίας ενημέρωσης, συμπεριλαμβανομένης της επικοινωνίας GitHub από αυτή την άποψη.

## Εξυπηρετητής ASF

Το ASF επικοινωνεί με **[τον δικό μας διακομιστή](https://asf.justarchi.net)** για πιο προηγμένη λειτουργικότητα. Ειδικότερα, αυτό περιλαμβάνει:
- Επαλήθευση των αθροισμάτων ελέγχου από το GitHub σε σχέση με τη δική μας ανεξάρτητη βάση δεδομένων για να διασφαλιστεί ότι όλες οι κατεβασμένες εκδόσεις είναι νόμιμες (χωρίς κακόβουλο λογισμικό), MITM επιθέσεις ή άλλες παραβιάσεις)
- Λήψη λίστας κακών bots για φιλτράρισμα αν έχετε κρατήσει `FilterBadBots` καθολική ρύθμιση ρυθμίσεων ενεργοποιημένη.
- Ανακοινώνοντας το bot σας στην **[καταχώρηση μας](https://asf.justarchi.net/STM)** αν έχετε ενεργοποιήσει `SteamTradeMatcher` σε **[`TradingPreferences`](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Configuration#tradingpreferences)** και πληρούν άλλα κριτήρια
- Κατεβάζοντας επί του παρόντος διαθέσιμα bots στο εμπόριο από την **[καταχώρησή μας](https://asf.justarchi.net/STM)** αν έχετε ενεργοποιήσει `MatchActively` σε **[`TradingPreferences`](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Configuration#tradingpreferences)** και πληρούν άλλα κριτήρια

Ως μέτρο ασφαλείας, δεν είναι δυνατόν να απενεργοποιήσετε την επαλήθευση αθροίσματος ελέγχου για ASF builds. Ωστόσο, μπορείτε να απενεργοποιήσετε εντελώς τις αυτόματες ενημερώσεις, αν θέλετε να το αποφύγετε αυτό, όπως περιγράφεται παραπάνω στην ενότητα GitHub.

Μπορείτε να απενεργοποιήσετε τη ρύθμιση `FilterBadBots` αν θέλετε να αποφύγετε τη λήψη της λίστας από το διακομιστή.

Μπορείτε να αποφασίσετε να εξαιρεθείτε από την ανακοίνωση στην καταχώρηση απενεργοποιώντας τη σημαία `PublicListing` στο **[`RemoteCommunication`](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Configuration#remotecommunication)** ρυθμίσεις. Αυτό μπορεί να είναι χρήσιμο αν θέλετε να εκτελέσετε το `SteamTradeMatcher` bot χωρίς να ανακοινωθεί ταυτόχρονα.

Η λήψη bots από την καταχώρησή μας είναι υποχρεωτική για τη ρύθμιση `MatchActively` , θα πρέπει να απενεργοποιήσετε αυτή τη ρύθμιση αν δεν είστε πρόθυμοι να την αποδεχθείτε.
