# Πρόσθετα

Το ASF περιλαμβάνει υποστήριξη για προσαρμοσμένα πρόσθετα που μπορούν να φορτωθούν κατά τη διάρκεια του χρόνου εκτέλεσης. Τα πρόσθετα σας επιτρέπουν να προσαρμόσετε τη συμπεριφορά του ASF, για παράδειγμα προσθέτοντας προσαρμοσμένες εντολές, προσαρμοσμένη λογική συναλλαγών ή πλήρη ενσωμάτωση με υπηρεσίες και API τρίτων μερών.

Αυτή η σελίδα περιγράφει τα πρόσθετα ASF από την προοπτική των χρηστών - εξήγηση, χρήση και το ίδιο. Αν θέλετε να δείτε την προοπτική του προγραμματιστή, μετακινήστε **[εδώ](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Plugins-development)**.

---

## Χρήση

Το ASF φορτώνει plugins από τον κατάλογο `plugins` που βρίσκεται στο φάκελο ASF σας. Είναι μια συνιστώμενη πρακτική (η οποία γίνεται υποχρεωτική με αυτόματες ενημερώσεις plugins) για να διατηρήσει ένα αποκλειστικό κατάλογο για κάθε plugin που θέλετε να χρησιμοποιήσετε, που μπορεί να βασιστεί στο όνομά του, όπως το `MyPlugin`. Με αυτόν τον τρόπο θα προκύψει η τελική δομή δέντρου του `plugins/MyPlugin`. Τέλος, όλα τα δυαδικά αρχεία του plugin θα πρέπει να τεθεί μέσα σε αυτό το ειδικό φάκελο, και ASF θα ανακαλύψουν και να χρησιμοποιήσετε το plugin σας μετά την επανεκκίνηση.

Usually plugin developers will publish their plugins in form of a `zip` file with binaries inside, which means that you should unpack that zip file to its own dedicated subdirectory inside `plugins` directory.

Αν το πρόσθετο φορτωθεί με επιτυχία, θα δείτε το όνομα και την έκδοσή του στο αρχείο καταγραφής σας. Θα πρέπει να συμβουλευτείτε τους προγραμματιστές plugin σας σε περίπτωση ερωτήσεων, θεμάτων ή χρήσης που σχετίζονται με τα plugins που έχετε αποφασίσει να χρησιμοποιήσετε.

Μπορείτε να βρείτε μερικά προτεινόμενα πρόσθετα στην ενότητα **[τρίτων](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Third-party#asf-plugins)**.

**Παρακαλώ σημειώστε ότι τα πρόσθετα ASF θα μπορούσαν να είναι κακόβουλα**. Θα πρέπει πάντα να διασφαλίζετε ότι χρησιμοποιείτε plugins που γίνονται από προγραμματιστές που μπορείτε να εμπιστευτείτε, ακόμη και από την ενότητα τρίτου μέρους παραπάνω. Οι προγραμματιστές ASF δεν μπορούν πλέον να σας εγγυηθούν τα συνήθη οφέλη ASF (όπως η έλλειψη κακόβουλου λογισμικού ή η απουσία VAC) εάν αποφασίσετε να χρησιμοποιήσετε οποιαδήποτε προσαρμοσμένα plugins. Πρέπει να καταλάβετε ότι τα plugins έχουν τον πλήρη έλεγχο της διαδικασίας ASF μόλις φορτωθούν, λόγω αυτού δεν μπορούμε επίσης να υποστηρίξουμε ρυθμίσεις που χρησιμοποιούν προσαρμοσμένα πρόσθετα, αφού δεν τρέχετε πλέον κωδικό ASF της βανίλιας.

## Συμβατότητα

Ανάλογα με την πολυπλοκότητα, το πεδίο εφαρμογής και πολλούς άλλους παράγοντες, είναι εντελώς πιθανό ότι θα απαιτήσει από εσάς να χρησιμοποιήσετε την παραλλαγή **[γενικού](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Setting-up#generic-setup)** ASF, αντί να συνιστάται συνήθως **[OS-συγκεκριμένο](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Setting-up#os-specific-setup)**. Αυτό οφείλεται στο γεγονός ότι η συγκεκριμένη παραλλαγή OS-έρχεται μόνο με βασική λειτουργικότητα που απαιτείται για το ίδιο το ASF, and your plugin may require parts that fall outside of main ASF scope, in result being incompatible with trimmed OS-specific builds.

Σε γενικές γραμμές, όταν χρησιμοποιείτε πρόσθετα τρίτων, συνιστούμε να χρησιμοποιείτε τη γενική παραλλαγή ASF για μέγιστη συμβατότητα. Ωστόσο, όχι όλα τα plugins μπορεί να το απαιτήσουν - παρακαλούμε ανατρέξτε στις πληροφορίες του plugin σας για να μάθετε αν πρέπει να χρησιμοποιήσετε γενική παραλλαγή ASF ή όχι.

## Αυτόματες ενημερώσεις

Το ASF έχει ενσωματωμένο μηχανισμό για αυτόματες ενημερώσεις προσθέτων. Για να λειτουργήσει αυτό το χαρακτηριστικό, πρώτα απ 'όλα, plugin της επιλογής σας πρέπει να **[υποστήριξη](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Plugins-development#automatic-updates)** αυτόν τον μηχανισμό. Αν έχετε φορτώσει ένα plugin που υποστηρίζει αυτόματες ενημερώσεις, ASF θα αναφέρει στο αρχείο καταγραφής κατάλληλα κατά την προετοιμασία του plugin, όπως "plugin έχει απενεργοποιηθεί από αυτόματες ενημερώσεις" ή "plugin έχει καταχωρηθεί και ενεργοποιηθεί για αυτόματες ενημερώσεις".

Από προεπιλογή, οι αυτόματες ενημερώσεις για προσαρμοσμένα πρόσθετα είναι **απενεργοποιημένες**, για λόγους ασφαλείας. Μπορείτε να ρυθμίσετε τις αυτόματες ενημερώσεις στο config χρησιμοποιώντας **[`PluginsUpdateList`](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Configuration#pluginsupdatelist)** και/ή **[`PluginsUpdateMode`](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Configuration#pluginsupdatemode)**, συνιστούμε να διαβάσετε την περιγραφή αυτών των ιδιοτήτων config για περισσότερες πληροφορίες. Είναι επίσης ωραίο να σημειωθεί ότι, όπως και με τις ενημερώσεις ASF, μπορείτε να αποφασίσετε να κρατήσετε τις αυτόματες ενημερώσεις απενεργοποιημένες, και στη συνέχεια να ενημερώσετε σχετικά με όπως απαιτείται, χειροκίνητη βάση, εκδίδοντας εντολή `updateplugins` **[](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/Commands)**.

Και οι δύο προσεγγίσεις σας επιτρέπουν να ενημερώσετε κανένα, μερικά, ή όλα τα προσαρμοσμένα plugins που έχετε φορτώσει στη διαδικασία.
