# Terceiros

Essa seção inclui vários aditamentos de terceiros escritos exclusivamente (ou principalmente) para serem usados em conjunto com o ASF. Elas variam desde plugins do ASF, até simples aplicações web e bibliotecas internas para integração, terminando com bots cheios de recursos para as outras plataformas. Se você gostaria de adicionar algo à lista, conte-nos no Discord ou em nosso grupo Steam.

Please note that below programs are **not** maintained by ASF developers and therefore **we give no guarantee about any of them**, especially in terms of security, safety or Steam ToS compliance. Esta lista é mantida apenas para referência. Você deve sempre garantir que o utilitário de terceiros que você está prestes a usar é seguro e legível o suficiente para você já que você está usando todos eles por sua conta e risco.

## Plugins ASF

### **[CatPoweredPlugins](https://github.com/CatPoweredPlugins)** (**[Rudokhvist](https://github.com/Rudokhvist)**)

- **[ASFAchievementManager](https://github.com/CatPoweredPlugins/ASFAchievementManager)**, plugin que te permite gerenciar conquistas Steam.
- **[Aniversário](https://github.com/CatPoweredPlugins/BirthdayPlugin)**, plugin para o ASF receber felicitações pelo aniversário.
- U**[CaseInsensitiveASF](https://github.com/CatPoweredPlugins/CaseInsensitiveASF)**, plugin para fazer nomes de bots sem distinção entre maiúsculas e minúsculas.
- **[CommandAliasPlugin](https://github.com/CatPoweredPlugins/CommandAliasPlugin)**, plugin que configura comandos personalizados para comandos do ASF.
- **[CommandlessRedeem](https://github.com/CatPoweredPlugins/CommandlessRedeem)**, plugin para o ASF reimplementar a chave resgatada sem um comando.
- **[ItemDispenser](https://github.com/CatPoweredPlugins/ItemDispenser)**, plugin que aceita automaticamente a solicitação de troca de determinado tipo de itens.
- **[SelectiveLootAndTransferPlugin](https://github.com/CatPoweredPlugins/SelectiveLootAndTransferPlugin)**, plugin que fornece o comando `avançado da transferência` por transferir itens Steam.

### **[Citrinato](https://github.com/Citrinate)**

- **[BoosterManager](https://github.com/Citrinate/BoosterManager)**, plug-in para o ASF fornece uma interface fácil de usar para transformar gemas em pacotes de cartas e vários recursos para gerenciar inventários e listagens de mercado.
- **[CS2Interface](https://github.com/Citrinate/CS2Interface)**, plugin para o ASF que permite a você interagir com Counter-Strike 2 usando **[IPC](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/IPC)**.
- **[FreePackages](https://github.com/Citrinate/FreePackages)**, plugin para o ASF que encontra pacotes gratuitos no Steam e os adiciona à sua conta.

### **[Vita](https://github.com/ezhevita)**

- **[Aceita de](https://github.com/ezhevita/FriendAccepter)**, plugin que permite que o ASF aceite automaticamente todos os convites de amizade.
- **[GameRemover](https://github.com/ezhevita/GameRemover)**, plugin que implementa um comando para remover as licenças Steam dos bots selecionados.
- **[GetEmail](https://github.com/ezhevita/GetEmail)**, plugin para o ASF que implementa um comando para buscar o endereço de e-mail dos bots indicados diretamente do Steam.
- **[ResetAPIKey](https://github.com/ezhevita/ResetAPIKey)**, plugin que implementa um comando para redefinir a chave API para os bots selecionados.

### Outros

- **[ASFEnhance](https://github.com/chr233/ASFEnhance)**, plugin que o fortalece com várias funcionalidades novas, especialmente comandos.
- **[ASFFreeGames](https://github.com/maxisoft/ASFFreeGames)**, plugin que permite que um colete automaticamente jogos Steam publicados no reddit.

## Integrações

- **[ASFBot](https://github.com/dmcallejo/ASFBot)**, bot do telegram escrito em python com integração com o ASF.
- **[ASF STM userscript](https://greasyfork.org/en/scripts/404754-asf-stm)**, para aqueles que desejam enviar ofertas de troca automatizadas para bots em nosso **[ASF STM listando](https://github.com/JustArchiNET/ArchiSteamFarm/wiki/ItemsMatcherPlugin#publiclisting)** pelo navegador da web, sem usar o recurso `MatchActive` fornecido pelo ASF.
- **[simple-asf-bot](https://github.com/deluxghost/simple-asf-bot)**, outro bot de telegram (mínimo) escrito em python com integração com o ASF.

## Embalagem

- **[Repositório no AUR #2](https://aur.archlinux.org/packages/archisteamfarm-bin)**, permitindo que você instale facilmente o ASF no Arch linux.
- **[Homebrew](https://formulae.brew.sh/formula/archi-steam-farm)**, permitindo que você instale facilmente o ASF no macOS.
- **[Nix](https://search.nixos.org/packages?channel=unstable&show=ArchiSteamFarm&from=0&size=50&sort=relevance&type=packages&query=ArchiSteamFarm)**, permitindo que você instale facilmente o ASF nas distribuições com Nix.
- **[NixOS](https://search.nixos.org/options?channel=unstable&from=0&size=50&sort=relevance&type=packages&query=ArchiSteamFarm)**, permitindo que você configure e instale o ASF com NixOS.
- **[Scoop](https://scoop.sh/#/apps?q=ArchiSteamFarm)**, permitindo que você instale facilmente o ASF nas janelas.
- **[Winget](https://github.com/microsoft/winget-pkgs/tree/master/manifests/j/JustArchiNET/ArchiSteamFarm)**, permitindo que você instale facilmente o ASF nas janelas.

## Ferramentas

- **[CS2Script](https://github.com/Citrinate/CS2Script)**, permite que você gerencie suas unidades de armazenamento Counter-Strike 2 com ajuda do plugin **[CS2Interface](https://github.com/Citrinate/CS2Interface)**.
- **[Keys extrator](https://umaim.github.io/SKE)**, permite que você copie e cole as keys em vários formatos e cria o comando `redeem` para o ASF. Confira **[repositório do GitHub](https://github.com/PixvIO/SKE)** para mais detalhes.
- **[ASF Mass Config Editor](https://github.com/genesix-eu/ASF_MCE)**, o que permite gerenciar várias configurações do ASF mais facilmente.
